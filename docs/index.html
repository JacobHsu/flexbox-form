根據標準，兩個控件都是行內塊級元素（inline-block），也就是說，它們默認並排在一行上。

這兩個控件之間有3~4像素的間隔，這是瀏覽器的內置樣式指定的。

指定表單使用 Flexbox 佈局。 兩個控件之間的間隔消失了，因為彈性佈局的項目（item）默認沒有間隔。

兩個控件元素的寬度沒有發生變化，因為彈性佈局默認不改變項目的寬度。

輸入框佔據當前行的所有剩餘寬度，只需要指定輸入框的flex-grow屬性為1

按鈕變高了，輸入框也自動變得一樣高

彈性佈局默認不改變項目的寬度，但是它默認改變項目的高度。如果項目沒有顯式指定高度，就將佔據容器的所有高度。

input align-self屬性可以改變高度行為。

如果項目很多，一個個地設置align-self屬性就很麻煩。這時可以在容器元素（本例為表單）設置align-items屬性

form元素設置了align-items以後，就不用在控件上設置align-self
